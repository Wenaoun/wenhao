# 墨迹 · 本地笔记

一款简洁优雅的本地笔记工具，数据完全保存在浏览器 localStorage 中。

## 功能特色

- **Markdown 编辑** — 完整语法支持，工具栏快捷插入，编辑/分屏/预览三种视图
- **标签系统** — 多标签分类，5 种颜色自动分配，标签筛选
- **全文搜索** — 搜索标题、内容、标签
- **数据管理** — localStorage 本地保存，支持 JSON 全部导出、Markdown 单篇导出、JSON/MD/TXT 导入
- **快捷操作** — 右键菜单复制/置顶/导出/删除，快捷键支持
- **明暗主题** — 一键切换，偏好自动保存
- **响应式布局** — 适配移动端

## 使用方式

直接用浏览器打开 `index.html` 即可使用，无需任何服务器或构建步骤。
